## Run Locally

Clone the project

```bash
  git clone https://github.com/karanmittal09/E-Commerce-web.git
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
